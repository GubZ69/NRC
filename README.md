# NRC
   

NUMBER ROW CHARACTERS
   
   ! @ \# $ % ^ & * \( \) _ + 

- !  - Exclamation Mark
- @  - At Sign
- \# - Number Sign
- $  - Dollar Sign
- %  - Percentage Sign
- ^  - Circumflex
- &  - Ampersand
- \*  - Astrix
- \( - Left Parenthesis
- \) - Right Parenthesis
- \_ - Underscore
- \+ - Plus Sign
